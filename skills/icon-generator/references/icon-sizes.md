# Icon size tables

## Web UI/UX (favicons + PWA)

Minimal, modern set:

- `favicon.ico`: 16×16, 32×32 (optionally 48×48)
- `apple-touch-icon.png`: 180×180
- `icon-192.png`: 192×192
- `icon-512.png`: 512×512
- `icon-maskable-512.png`: 512×512 (opaque background + padding)

## Unreal Engine packaging (optional)

### Windows `.ico`

Recommended embedded sizes:

- 16, 24, 32, 48, 64, 128, 256

### macOS `.iconset` (source for `.icns`)

Common iconset sizes and filenames:

- `icon_16x16.png` (16)
- `icon_16x16@2x.png` (32)
- `icon_32x32.png` (32)
- `icon_32x32@2x.png` (64)
- `icon_128x128.png` (128)
- `icon_128x128@2x.png` (256)
- `icon_256x256.png` (256)
- `icon_256x256@2x.png` (512)
- `icon_512x512.png` (512)
- `icon_512x512@2x.png` (1024)

### Linux

- 256×256 PNG (common baseline)
- 512×512 PNG (nice to have)
